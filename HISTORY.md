# IKSZ Project History

This document records the historical timeline of "Igen Konnyu Szerepjatek
(IKSZ)" project from 1997 to 2003, reconstructed from website archives,
database records, and original PDF files.

---

## 1997-2000: Foundation Period

### 1997
- IKSZ project founded
- Development of core rulebook begins
- Copyright established: "Az Igen Konnyu Szerepjatek alkotoi"

### 1998-1999
- Core rules development continues
- Early world concepts developed
- Team collaboration establishes modular philosophy

### February 14, 2000
- Terido (Sci-Fi world) module published
- Author: Priest (Varga Jozsef)
- Format: HTML
- Description: Sci-Fi vilag

### May 22, 2000
- Reika (Fantasy world) module published
- Author: Kosa Laszlo (klinsi)
- Format: HTML
- Description: "Szep uj vilag" - Fantasy vilag

### May 30, 2000
- Doraddon kiegeszitesek published
- Author: Erfalvy Aron (E.y.A)
- Format: TEXT
- Description: "Egy halvany mecses a sotetsegben" - Doraddon vilagszemlelet es kalandotletek

### July 17, 2000
- Major website relaunch at iksz.hu
- New infrastructure: MySQL database, PHP engine, Apache server
- Website redesign completed
- Author: Nagy Balazs (julian7)

### July 30, 2000
- Novels section launched on website
- First novella published: "A hajo" (The Ship)
- Author: Nagy Balazs
- Setting: Doraddon
- Call for community contributions to write novellas

### July 31, 2000

- Users can rate modules and novellas

### August 2, 2000
- OpenRPG 0.4.3a multiplatform online RPG released
- IKSZ community begins exploring online play options

### September 29, 2000
- Server migration issues
- Temporary domain problems with kva.hu

### October 4, 2000
- Successful server migration completed
- New location: Victor Hugo utca (BIX datacenter)
- Connection: 10 Mbps (planned upgrade to 100 Mbps)
- International bandwidth: 128 kbps

### October 12, 2000
- Ineria PDF updated
- Bug fixes and typography improvements
- Preparation for Ineria novella

### October 31, 2000
- New server hardware installed
- Infrastructure stabilization
- Focus shifts back to IKSZ content development

### November 2, 2000
- All PDFs regenerated with proper fonts
- Typography and style improvements across all modules
- Second novella in progress

### November 9, 2000
- Multiple module releases:
  - karakterlap (Character sheet) PDF published
  - Doraddon PDF v1.0 published
  - Merano PDF v1.0 published (draft status)
  - IKSZ Kialtvany (Manifesto) PDF published

### November 12, 2000
- Major releases:
  - IKSZ core rulebook updated
  - Ineria PDF v1.0 published
  - "A hajo" novella PDF version published
- Core rulebook download count: 178
- Ineria download count: 55
- Doraddon download count: 54

### November 13, 2000

  - Setting: Ineria
  - Length: 54 KB (approximately 1/6 of a book)

  - Miracle (Termeszetfeletti) supplement published

  - Download count: 58

  - Fegyverek es felszerelesek (Weapons & Equipment) supplement published

  - SABLON (Template) module published
  - Two formats: PDF and ZIP (LaTeX sources)
  - Purpose: Template for community module creation

  - Community poll launched: "Szerinted jatszhat az IKSZ?"
    (Do you think IKSZ is playable?)

## Technical Infrastructure

### Website Evolution
- Original hosting: kva.hu domain
- Migrated to: iksz.hu (July 2000)
- Server location: Victor Hugo utca, Budapest (BIX datacenter)
- Technology stack: Apache, PHP, MySQL
- Bandwidth: 10 Mbps local, 128 kbps international

### Distribution
- Primary format: PDF (generated from LaTeX)
- Secondary formats: HTML, TEXT
- Source format: LaTeX (SABLON template provided)
- Download tracking: Implemented August 2000
- Community rating: Implemented August 2000

## Design Philosophy

From the modules.php source code (2000):

"Az IKSZ egyik alappillere az, hogy konnyu. Ezert nem kivanunk minden kis
nuansznyi dolgot bo lere ereszteni, es feltételként szabtuk, hogy egy modul
terjedelme 20 oldalban megall."

Translation: "One of IKSZ's core principles is that it's easy. Therefore, we
don't want to elaborate on every little nuance, and we set a requirement that
a module's length should fit within 20 pages."
