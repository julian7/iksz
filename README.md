# IKSZ Restoration Project

**Igen Könnyű Szerepjáték (IKSZ) - Hungarian Tabletop RPG System**

This repo contains the restored artifacts of the legacy ConTeXt sources converted to Markdown format, put into a publishing suite to generate PDF, HTML, EPub, and Mobi formats of the original documents.

## What's Been Converted

- IKSZ: Core Rulebook
- IKSZ: Core Rulebook - earlier version
- Természetfeletti: magic expansion
- Fegyverek: weapons and combat system
- World Modules:
  - Doraddon: dark fantasy world
  - Ineria: high fantasy world
  - Merano: dark future world - draft
  - Reika: fantasy world
  - Téridő: Sci-fi world
- Novellas:
  - A Hajó (The Ship) - Doraddon setting
  - Az Álomlátó (The Dream-seer) - Ineria setting
  - Ismerd meg ellenfeled (Know your Enemy) - Cyberpunk setting
  - A Fényhozó legendája: A gyermek (Legend of Lightbringer: The Child) - Donaddon setting
